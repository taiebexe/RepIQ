# RepIQ

AI-powered workout analytics dashboard for [Hevy](https://www.hevyapp.com/) data. Fetches your workout history, computes training analytics, and uses Claude AI to generate personalized coaching insights.

---

## Features

- **Dual Authentication** — Log in with Hevy credentials (free, full data access) or a Hevy PRO API key
- **Dashboard Analytics** — Workout count, total volume, training streaks, session duration, frequency
- **Lift Progression** — Estimated 1RM tracking over time for your top 6 exercises with area charts
- **Weekly Volume** — Stacked bar chart showing sets per muscle group across 8 weeks
- **Training Frequency** — 90-day GitHub-style heatmap of workout activity
- **Plateau & Strength Detection** — Flags exercises where progress has stalled or declined
- **Recent PRs** — Personal records pulled directly from your workout data (credentials login)
- **AI Coach Insights** — 3 personalized, actionable insights generated by Claude based on your actual numbers
- **Fully Stateless** — No database, no user accounts. Auth tokens live in sessionStorage only.

## Login Methods

| Feature | Hevy Credentials | Hevy PRO API Key |
|---|---|---|
| Requires PRO | No | Yes |
| Volume Data | Full (estimated by Hevy) | Calculated from sets |
| Muscle Groups | From API | Local mapping |
| Personal Records | Yes | Limited |
| Data Range | Last 90 days | Last 90 days |

> **Recommended:** Use Hevy Credentials for the richest data.

## Getting Started

### Prerequisites

- Node.js 18+
- An [Anthropic API key](https://console.anthropic.com/) (for AI insights, optional)

### Installation

```bash
git clone <repo-url>
cd RepIQ
npm install
```

### Configuration

Create a `.env.local` file in the project root:

```
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

AI insights will use fallback messages if this key is not set.

### Running

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and log in with your Hevy credentials or API key.

## Privacy

- Hevy credentials are sent directly to Hevy's servers and never stored
- API keys and tokens live in `sessionStorage` only (cleared when the tab closes)
- No data is persisted server-side
- The Anthropic API key is server-side only (`.env.local`)
