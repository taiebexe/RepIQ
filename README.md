# RepIQ

AI-powered workout analytics dashboard for [Hevy](https://www.hevyapp.com/). Log in with your Hevy account, and RepIQ fetches your entire workout history, computes detailed training analytics, and uses Claude AI to generate personalized coaching insights.

No database. No accounts. Fully stateless.

---

## Features

### Dashboard Analytics
- **6 summary stats** — Total workouts, volume lifted, total sets, training streak, avg session duration, weekly frequency
- **Muscle Distribution** — Donut chart showing set breakdown by muscle group
- **Muscle Balance** — Radar chart visualizing training balance across all muscle groups
- **Lift Progression** — Estimated 1RM area chart with progress % for your top 6 exercises
- **Weekly Volume** — Stacked bar chart of sets per muscle group over 8 weeks
- **Volume Trend** — 12-week combo chart (volume bars + sets line)
- **Session Duration** — Duration trend over time with average reference line
- **Rep Range Distribution** — Breakdown of strength (1-5), hypertrophy (6-12), endurance (13-20), and high rep (21+) sets
- **Workout Time** — Color-coded bar chart showing what time of day you train
- **Training Frequency** — Full-year GitHub-style heatmap
- **Exercise Ranking** — Table of top 12 exercises with sessions, volume, best 1RM, and trend arrows
- **Personal Records** — Timeline of recent PRs grouped by date
- **Plateau & Strength Detection** — Flags exercises as plateaued, declining, or gaining
- **AI Coach Insights** — 3 specific, actionable insights generated by Claude based on your actual numbers

### Authentication
- **Hevy Credentials** — Log in with your Hevy email/username and password (no PRO required, full data access)
- **Hevy PRO API Key** — Use your API key from [hevy.com/settings?developer](https://hevy.com/settings?developer)

### Data
- Fetches your **complete workout history** (all time, not just 90 days)
- All computation happens server-side in Next.js API routes
- Nothing is stored — auth tokens live in `sessionStorage` only

## Getting Started

### 1. Prerequisites

- **Node.js 18+** — check with `node -v`
- **AI API key** (optional) — for AI coaching insights. **Gemini is free** at [aistudio.google.com/apikey](https://aistudio.google.com/app/apikey). Also supports [Anthropic](https://console.anthropic.com/) and [OpenAI](https://platform.openai.com/)

### 2. Install

```bash
git clone <repo-url>
cd RepIQ
npm install
```

### 3. Configure (optional)

Create a `.env.local` file in the project root to enable AI insights:

```env
# Free — get a key at https://aistudio.google.com/apikey
GEMINI_API_KEY=your-gemini-key-here

# Optional paid providers (tried if Gemini is not set):
# ANTHROPIC_API_KEY=sk-ant-your-key-here
# OPENAI_API_KEY=sk-your-key-here
```

If you skip all, the dashboard will show generic fallback insights instead of personalized ones.

### 4. Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 5. Log in

**Option A — Hevy Credentials (recommended):**
1. Enter your Hevy email/username and password
2. Click **Log in & Analyze**
3. Wait for your workout data to load

**Option B — API Key:**
1. Switch to the **API Key** tab
2. Get your key from [hevy.com/settings?developer](https://hevy.com/settings?developer) (requires Hevy PRO)
3. Paste it and click **Connect & Analyze**

The dashboard loads automatically after login. Click **Disconnect** in the top-right to log out.

## Privacy & Security

- Your Hevy credentials are sent **directly to Hevy's servers** via their API. RepIQ never stores them.
- Auth tokens live in `sessionStorage` — they are automatically cleared when you close the browser tab.
- No data is saved to disk, database, or any external service.
- The Anthropic API key is server-side only (never exposed to the browser).
- All API calls to Hevy are proxied through Next.js API routes so your tokens never leave the server.
